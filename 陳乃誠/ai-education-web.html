一、NotebookLM 工具介紹

NotebookLM核心定位

NotebookLM定位為研究學習必備工具，最大特色是「有所本」—以文件為本，回答始終依據您上傳的資料，確保引用有據可循。

NotebookLM與ChatGPT比較

NotebookLM ChatGPT
專注於用戶的文件資料 基於通用網絡知識
可追蹤引用來源 無法提供精確引用
適合深度研究分析 適合廣泛話題討論
引用有明確頁碼段落 資訊可能未經驗證

AI幻覺問題與應對

陳校長開發了輔助教師設計AI防弊教學活動的助手，此工具能協助：

  • 教師設計「陷阱教材」
  • 提供學生練習幻覺辨識的對話情境
  • 點評學生交來的AI生成文字，分析其幻覺風險點

NotebookLM功能比喻

NotebookLM就像是「超級圖書館員 + 聰明作家」的組合：

  • 圖書館員（檢索）：當你提問，它會在你的記事和上傳的來源中快速翻找，找到最相關的信息
  • 聰明作家（生成）：不是單純複製內容，而是理解資訊後，用流暢自然的語言，幫你總結重點或解釋概念
走進圖書館，告訴館員：「我想了解 XX」
他不僅迅速找到對應的書籍，還能幫你濃縮內容
還能用更容易理解的方式重新表達，讓你輕鬆獲取有用資訊，而不用自己翻遍所有資料

NotebookLM操作特點

操作很簡單，但應用才精彩。提供使用者友善的介面和直觀的操作方式。

NotebookLM介面與功能詳解

可建立至少100本筆記本(Notebook)，每本筆記本內有三個主要區塊：

1. 來源(Sources)

  • 來源資料格式多元，支援多種檔案類型
  • 每個筆記本最多可包含50個sources
  • 每個source最多500,000字，意味一個筆記本可處理高達2500萬字內容
  • 資料上傳後自動進行處理和索引

2. 對話(Chat)

  • 「勾選」代表現在正在和哪些sources/notes進行對話互動
  • 系統生成3個建議對話提問，引導使用者開始交談
  • 對話結果可「儲存至記事」，方便整理和保存重要內容
  • Plus版本可自訂對話回覆風格（例如學術、簡潔、詳細等）與回覆長度
  • 支援連續對話，保持上下文理解

3. Studio區域

  • 含記事Notes：保存和整理重要內容
  • 指令Commands：提供特定功能操作
  • 語音摘要：將內容轉為音頻形式
  • 指令功能詳細說明：
    • 內建「簡介文件」：快速生成文件概述
    • 「研讀指南」：提供系統化的學習建議
    • 「常見問題」：自動生成關於內容的FAQ
    • 「時間軸」：生成事件時序脈絡
    • 所有生成結果均以記事形式保存
  • 記事可「轉換成來源」，實現內容的循環利用

二、NotebookLM實際應用案例

應用1：私人法務顧問

法規遵循 行政管理 採購法規

情境描述

"我是8月甫上任的總務主任，面對採購法規仍不熟悉，我的庶務組長也是新手，深怕不小心違反採購法引發爭議。"

實際應用

上傳法規文件，檢查投標須知的合法性

運作機制

  1. 上傳相關法律文件（如採購法規）
  2. 上傳待檢查的招標文件
  3. AI結合兩者進行分析比對
  4. 生成符合法規的修正建議

實例展示

檢查英越語營投標須知的問題點：

  • 具體指出須知中第48-1條「植栽工程養護期保證金」問題
  • 指出「押標金保證金暨其他擔保作業辦法」相關條文編號錯誤
  • 提供條款正確引用建議
  • 每項建議皆附有法規依據，確保修正符合最新法規要求

應用2：跨域、跨時空激盪跨界想法

創意思考 跨領域學習 歷史比較

情境主題

當蘇東坡遇見賈伯斯——跨越千年的思想對話

執行方法

  1. 把描述兩人的作品、傳記上傳到NotebookLM
  2. 用對話功能探問兩人之間的觀點異同

研究問題示例

  • 蘇東坡與賈伯斯有哪些人格特質或生涯遭遇的相似處？
  • 兩人面對挫折的態度有何異同？
  • 他們對創新與傳統的看法如何比較？

分析結果

系統整理出兩位跨時代創新者的共同特質：

  • 才華橫溢、不拘束縛
  • 遭遇挫折時的彈性與復原力
  • 跨領域思維與創造力

創意應用

模仿蘇東坡寫作風格改寫「青蛙爬山」故事，展現古典意境，加入東坡詩詞風格的句式與意境

教育價值

培養跨領域思考能力，理解不同時代創新者的共通思維

應用3：分析歷屆試題

教學研究 考試分析 命題趨勢

情境

分析國中教育會考社會科命題趨勢

問題改進技巧

透過改寫與擴寫任務說明，提升AI回答品質

原始提問

「從歷年試題來看，國中教育會考社會科命題趨勢為何」

改寫擴寫後提問

包含六大面向的詳細分析要求：

  1. 命題主題演變：檢視近五年或更長時間的考題變化趨勢
  2. 試題類型：比較不同題型比重與變化
  3. 素養導向趨勢：分析跨領域整合、生活情境題的發展
  4. 考試難度變化：評估高層次思考題目比例變化
  5. 時事與政策影響：探討與108課綱相關的命題趨勢
  6. 學生應試策略建議：基於趨勢分析提供準備建議

改進成效

  • 獲得更全面、系統化的分析結果
  • 分析結果包含數據支持，更有說服力
  • 便於教師掌握命題演變脈絡

實際應用案例

模擬命題實例：展示如何根據分析結果設計符合趨勢的公民科模擬試題，包含詳細解析與評分標準

應用4：用AI提問閱讀

閱讀策略 深度學習 問題導向

閱讀方式轉變

從「循序式閱讀」到帶著問題意識的「跳躍式閱讀」

資料範例

80K Hours、心經等深度閱讀材料

閱讀流程

1. 初步理解：利用Studio內建指令快速掌握文件框架
  • 「簡介文件」：獲取內容概述
  • 「研讀指南」：了解如何系統閱讀
  • 「常見問題」：掌握關鍵疑點
  • 「時間軸」：理解發展脈絡
2. 深度提問：針對有疑處提出不同層次問題

直接提問

例：「何謂五蘊皆空？」

簡單化提問

例：「請用國小五年級學生能理解的語言解釋五蘊皆空」

類比化提問

例：「請用類比方式解釋什麼是五蘊皆空？」

學習效益

  • 提高閱讀效率：無需逐字閱讀即可掌握關鍵內容
  • 增強理解深度：通過問答互動加深對核心概念的理解
  • 培養批判思考：從不同角度提問促進多維思考
  • 適應不同學習者：可根據學習者程度調整問題難度

應用5：更大脈絡下閱讀心智圖

視覺化學習 系統思考 知識關聯

功能說明

NotebookLM可自動生成資料的關聯視覺化

操作方法

  • 勾選多個相關來源文件
  • 系統自動分析文件間的關聯性
  • 生成互動式心智圖展現概念間連結

使用技巧

  • 點選心智圖上的任何節點可深入探索
  • 系統會顯示該概念在更大脈絡下的位置與關聯
  • 可放大縮小調整視圖範圍，聚焦特定主題

學習優勢

  • 系統化思考：將零散知識點整合成系統化知識網絡
  • 關聯性學習：理解概念之間的連結與影響關係
  • 整體性把握：同時獲得宏觀視角和微觀細節
  • 減少認知負荷：視覺化展示降低處理複雜資訊的負擔

教學應用

  • 協助學生建立學科知識框架
  • 輔助教師課程設計，確保知識點覆蓋全面
  • 支持跨學科整合教學，展示不同領域知識的連結點

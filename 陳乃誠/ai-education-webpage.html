AI引導語設計原則

設計AI引導語時需明確設定角色、目標、限制，並可進一步加入追問、格式與範例，以提升AI生成內容的精準度與實用性。

關鍵點

  • 設人設：給AI一個明確的身份角色。
  • 給目標：明確告知AI要完成的任務。
  • 給限制：設定具體要求，如數量、內容格式等。
  • 加追問：針對特定內容要求深入分析。
  • 加格式：如Markdown語法，方便後續編輯與應用。
  • 給範例：提供範例讓AI模仿、仿寫。

講者以國中會考作文批改、計畫審閱等實例，說明如何設計AI引導語，並強調用字精準度與格式要求的重要性。

設定AI為國中會考作文批改老師，要求針對圖片中的作文給予三個優點、兩個寫作建議及一個參考分數。

  • 設人設：你是國中會考作文批改老師。
  • 給目標：請針對圖片中的作文給予批改。
  • 給限制：請給三個優點、兩個建議、一個分數。
  • 加追問：請針對第二點建議給予深入分析。

注意事項

  • 用字需精準，避免模糊不清的指令。
  • 根據需求調整限制與格式，確保生成內容符合預期。
  • 提供範例有助於AI理解並仿寫。

特殊情況

  • 如果AI生成內容不符合需求，應檢查引導語是否明確並適時調整。
  • 如需特定格式，應明確告知AI格式要求。

設計有效AI引導語的六要素

設計AI引導語時需包含六個要素：設人設、給目標、給限制、加追問、加格式、給範例。

關鍵點

  • 設人設（如：你是唐代大詩人李白）
  • 給目標（如：針對圖片創作詩詞）
  • 給限制（如：用繁體中文台灣用語）
  • 加追問（如：請解析詩意）
  • 加格式（如：七言律詩）
  • 給範例（如：參考鄭愁予的詩）

教師強調設計AI引導語時，需明確設定人設、目標、限制、追問、格式與範例，才能讓AI生成符合需求的內容。

設定人設為李白，目標為針對三仙臺圖片創作七言律詩，限制為用繁體中文台灣用語，並要求解析詩意。

  • 明確設定人設與目標
  • 加入語言與格式限制
  • 要求AI進行內容解析
  • 可加入範例提升生成品質

注意事項

  • 重要資訊應寫在指令最前面
  • 繁體中文台灣用語需明確指定

特殊情況

若AI生成內容不符需求，檢查是否六要素皆已明確設定。

個資保護與匿名化處理

在將資料上傳AI進行分析前，需將個人資料（如姓名）刪除或以代號取代，避免個資外洩。

關鍵點

  • 將真實姓名改為代號（如1到30、66等）
  • 小孩的期末評語等資料也需匿名化
  • 即使AI平台聲稱不會使用資料，仍應謹慎處理

教師提醒，無論是問卷回饋或學生資料，皆需先將個資移除或改為代號後再上傳AI平台，保障資料安全。

收到回饋問卷時，將姓名等個資改為代號後再丟給AI分析。

  • 檢查資料中是否有個資
  • 將姓名等敏感資訊改為代號
  • 再將資料上傳AI平台進行分析

注意事項

  • 個資保護為首要原則
  • 即使平台宣稱安全，也不可掉以輕心

特殊情況

若資料涉及未成年學生，必須特別注意匿名化。

繁體中文台灣用語的重要性

在AI生成內容時，需明確指定使用繁體中文台灣用語，避免生成香港或其他地區語料。

關鍵點

  • 僅指定繁體中文可能導致語料不符台灣用語
  • 台灣用語需明確寫入指令
  • 不同地區語料差異大，影響內容品質

教師強調，AI平台的繁體中文語料多來自不同地區，若未指定台灣用語，可能生成不符合本地需求的內容。

在AI指令中加入「繁體中文台灣用語」八字，確保生成內容符合台灣語境。

  • 在指令中明確加入「繁體中文台灣用語」
  • 避免生成香港或其他地區語料
  • 提升內容本地化品質

注意事項

  • 每次生成文件、詩詞等內容時都需加上台灣用語
  • 避免語料不符導致內容不自然

特殊情況

若生成內容出現不符合台灣用語，檢查指令是否有明確指定。

  • 生成內容時需耐心等待（如網頁約五分鐘）
  • 每日免費生成簡報次數為三次，需妥善規劃使用
  • 部分內容因版權限制無法下載，需注意提示
  • 下載內容時需確認檔案格式（如圖片、PDF）

特殊情況

  • 如果遇到內容無法下載，可能因版權限制，請更換其他內容或樣式。
  • 如果生成內容後未顯示，請嘗試重新整理頁面或檢查歷史紀錄。
  • 如果簡報內容字體過小，可進入Canva進行進階編輯與調整。

Notion專案與任務管理

利用Notion進行專案與任務管理，將每日任務、專案、領域分開管理，並透過資料庫串接實現任務與專案的連動，方便追蹤與記錄。

關鍵點

  • 專案有明確期限，領域則無期限。
  • 每日任務可封存，未完成任務可暫存待處理。
  • 專案與任務資料庫可串接，任務完成後自動消失。
  • 可記錄每一天、每個專案的具體執行情況。
  • 知識管理與資源收集結合專案與領域。

講者以自身Notion管理流程為例，說明如何每日規劃任務、封存已完成事項，並將專案（如5G計畫、音樂會、領導力課程等）與無期限的領域（如個人健康、家庭旅行、資訊組等）分開管理。專案任務可設定Dayline，並與專案資料庫串接，確保資訊同步。

以5G計畫為例，專案內有第5次、第6次及期末報告，分別於2025-06-02、2025-06-11、2025-10-31執行。新增任務如『2025-05-20下午通知社群成員準備報告內容』，設定Dayline為2025-05-20，並歸屬於瑞士社群專案，身份為數學輔導團成員。任務完成後打勾即消失，未完成則持續提醒。

  • 在Notion中建立專案資料庫，輸入5G計畫相關任務。
  • 設定每個任務的Dayline與所屬專案。
  • 任務完成後打勾，資料庫自動更新。
  • 可查詢過往任務完成日期與所屬專案，方便回溯。

注意事項

  • 資料庫串接較複雜，需花時間學習。
  • 每日任務需持續維護，避免遺漏。
  • 專案與領域需明確區分，避免混淆。

特殊情況

  • 如果遇到任務未設Dayline，系統會自動提醒，需及時補充。
  • 如果專案或領域內容過多，建議只收集與自身成長或專案相關的資訊，避免資訊過載。

AI生成內容對教育與知識迭代的影響

AI工具可快速生成研究報告、小論文等內容，極大提升效率，但也導致知識迭代速度加快，對學生自學能力提出更高要求。

關鍵點

  • AI生成內容僅需1-2分鐘，遠快於傳統研究。
  • 知識迭代由原本10年縮短至2年，未來將更快。
  • 學生需培養自學能力，應對快速變化的知識環境。

講者指出，AI工具如Notebook LN可在2分鐘內生成一篇小論文，對比傳統需一週，知識產出與迭代速度大幅提升。未來學生面對的知識環境將更加快速變化，學校教育應重視自學能力的培養。

注意事項

  • AI生成內容可能削弱學生自主學習與思考能力。
  • 教育應調整重點，強化自學與批判性思考訓練。

特殊情況

如果學校舉辦小論文活動，AI工具可大幅縮短學生準備時間，需重新思考評量方式。

AI聲音複製技術的風險與防範

AI技術可於15秒內複製聲音，詐騙者可利用此技術進行聲音模仿，需提高警覺。

關鍵點

  • 15秒即可複製聲音
  • 詐騙者可利用聲音模仿進行詐騙
  • 接到陌生電話時避免隨意發聲

現今AI技術進步，僅需15秒錄音即可複製聲音，詐騙者可用於冒充本人進行詐騙。建議接到陌生電話時不要隨意發聲，並可訓練切換聲線以防被模仿。

講者提醒現在詐騙者只需15秒即可複製聲音，過去需一分鐘錄音，現今技術大幅提升，需特別注意。

  • 詐騙者致電錄音15秒即可複製聲音。
  • 利用AI模仿進行詐騙。
  • 建議接到陌生電話時不要隨意發聲。

注意事項

  • 接到陌生電話時避免隨意發聲。
  • 提高警覺，防範聲音被複製。

特殊情況

如果懷疑聲音被複製，應立即通知相關單位並加強防範措施。

AI日曆助理Donut與Google行事曆整合

Donut是一款AI日曆助理，可與Google行事曆整合，透過LINE操作，支援語音輸入、拍照辨識文字、自動新增行程等功能。

關鍵點

  • 需先將Donut機器人加入LINE好友。
  • 在LINE輸入「設置」取得後臺連結，綁定Google帳號與日曆。
  • 可用語音、拍照或複製貼上方式新增行程。
  • 目前免費，未來可能收費。

老師示範如何將Donut機器人加入LINE，並綁定Google行事曆。可直接用語音或拍照新增行程，適合行政管理與時間安排。

老師用LINE語音輸入『下午4點半我要開車回竹光國中，進行核銷』，Donut自動將行程新增至Google行事曆。

  • 將Donut機器人加入LINE。
  • 在LINE輸入『設置』取得後臺連結，綁定Google帳號與日曆。
  • 用語音、拍照或貼上方式新增行程。
  • Donut自動將行程同步至Google行事曆。

注意事項

  • 需綁定Google帳號，若在意資安可僅用Donut本身日曆。
  • 目前免費，未來可能收費。

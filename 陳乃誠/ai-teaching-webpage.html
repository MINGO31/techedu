教師批改作文的心理與專業挑戰

教師在長期批改學生作文過程中，容易感受到心理壓力與專業能力的下滑，甚至會被學生的語言習慣同化，影響自身的文字能力。

關鍵點

  • 長期批改會導致腦力枯竭
  • 教師可能被學生的語言習慣同化
  • 專業能力可能因批改過程而下滑

講者分享自己在批改作文時的感受，認為長期下來會被學生的語言影響，甚至覺得自己的文字能力也會變差。

實例

講者表示改作文改到後來都覺得自己被學生拖垮了，自己的程度慢慢的會被同化。

  • 教師長期批改學生作文，會受到學生語言表達的影響。
  • 這種影響可能導致教師自身的文字能力下滑。

注意事項

  • 注意自身專業能力的維持
  • 適時調整批改方式，避免被學生語言同化

AI在國中作文批改的應用

利用AI協助國中老師進行作文批改，減輕老師批閱負擔，提升批改效率。AI可針對手寫辨識進行運用，雖然尚未達到百分之百精準，但已能協助老師完成部分批改工作。

關鍵點

  • AI可協助批改國中會考作文
  • AI能辨識手寫文字，雖未達百分之百精準，但已可運用
  • AI可給予作文優點、建議及參考分數
  • AI可減輕老師批閱負擔

講者分享自己在104年開始將資訊融入國文教學，至今已十年經驗。現今AI技術已能協助老師進行作文批改，尤其在手寫辨識方面，雖然尚未完全精準，但已能大幅減輕老師的工作負擔。AI可根據設定的人設、目標及限制，給予作文優點、建議及分數，並可進行追問以獲得更詳細的建議。

實例

設定AI為『臺灣國中會考的作文批改老師』，請AI針對圖片中的文章給予批閱，要求給出三個優點、兩個寫作建議及一個參考分數。如需更詳細建議，可追問AI針對某一點詳加說明。

  • 設定人設：你是臺灣國中會考的作文批改老師。
  • 設定目標：請針對圖片中的文章給予批閱。
  • 設定限制：請給三個優點、兩個寫作建議及一個參考分數。
  • 如需更詳細說明，進行追問。

注意事項

  • AI辨識手寫文字尚未百分之百精準，需人工檢查
  • AI批改建議需老師判斷其適用性
  • AI平台帳號權限可能影響功能開啟

特殊情況處理

  • 如果AI辨識結果不準確，老師需手動修正或補充。
  • 如果AI給出的建議不夠清楚，可進行追問要求更詳細說明。
  • 如果學校帳號AI功能未開啟，需聯絡管理員協助開啟。

AI平台padlet的操作與功能

padlet是一個支援AI功能的數位平臺，適合用於即時回饋、共編、上傳多媒體（影片、圖片、錄音、錄影）及AI生成圖片等教學活動。

關鍵點

  • padlet支援共用、共編功能
  • 可上傳影片、圖片、錄音、錄影
  • 內建AI文字生成圖片功能
  • 適合用於即時回饋與協作學習

講者介紹padlet平臺的操作方式，包括新增區段、輸入名字、繳交作業、AI生成圖片等。padlet平臺支援多種媒體上傳，並可即時回饋，適合老師進行分組協作與互助學習。

實例

老師在padlet平台新增區段，輸入名字，於名字下方按加號繳交作業。作業可包括AI生成圖片，並可發布於平台上供大家觀看。

  • 新增區段，輸入名字
  • 於名字下方按加號新增貼文
  • 選擇2.0模組，輸入指令生成圖片。
  • 確認生成內容為圖片而非照片。
  • 下載圖片至本地端。

注意事項

  • 避免指令中出現『照片』字樣
  • 根據使用頻率選擇模組

特殊情況處理

  • 如果生成內容為照片，需檢查指令是否正確。

圖片下載與上傳方式

Gemini生成的圖片可透過右上角下載鈕或點擊圖片右鍵另存圖片兩種方式下載，支援JPG與PNG格式。上傳作業時，於成果區選擇上傳鈕，點選圖片即可完成。

關鍵點

  • 下載方式：右上角下載鈕、右鍵另存圖片
  • 支援格式：JPG、PNG
  • 上傳方式：成果區加號→上傳鈕→選擇圖片

根據個人習慣選擇下載與上傳方式，確保圖片格式正確。

注意事項

  • 確認圖片格式為JPG或PNG
  • 下載後檢查檔案是否完整

特殊情況處理

  • 如果圖片無法上傳，檢查檔案格式是否為JPG或PNG。

對話框一對一工作原則

每個對話框僅處理一項工作，避免多項任務混雜於同一對話，便於日後查找與管理。

關鍵點

  • 一個對話處理一個任務
  • 新任務需開啟新對話
  • 對話紀錄自動保存

養成每個任務開新對話的習慣，有助於資料整理與後續查詢。

注意事項

  • 避免多項任務混用同一對話
  • 定期整理對話紀錄

特殊情況處理

  • 如果多項任務混雜於同一對話，日後查找會困難，建議重新分開。

Padlet練習與AI詩詞創作

將生成的圖片上傳後，設定AI人設（如唐代詩人李白），請AI根據圖片內容創作七言絕句並進行解析。國文老師需特別指定『繁體中文』與『臺灣用語』。

關鍵點

  • 設定AI人設與目標
  • 明確限制語言與用語
  • 要求AI創作並解析詩詞

透過人設與目標設定，提升AI生成內容的相關性與品質。

實例

人設：唐代大詩人李白，指令：依照圖片中的風景，寫一首七言絕句，用繁體中文呈現，並加以解析。

  • 上傳臺北101夜景圖片
  • 設定人設與語言限制
  • AI生成詩詞並附解析

注意事項

  • 明確指定語言與用語
  • 檢查AI生成內容是否符合要求

特殊情況處理

  • 如果AI未使用繁體中文或臺灣用語，需重新設定指令。

對話紀錄保存與管理

Gemini平台會自動保存所有對話紀錄，除非手動刪除。目前尚無資料夾分類功能，所有對話以無限格方式排列。

關鍵點

  • 對話自動保存
  • 無資料夾分類功能
  • 可手動刪除對話

需自行管理對話數量，避免過多對話造成查找困難。

注意事項

  • 定期整理與刪除不需要的對話

特殊情況處理

  • 如果對話過多難以查找，建議手動刪除或標記重點。

複製與分享對話內容

關鍵點

  • 複製功能保留格式（如markdown）

  • 學校帳號需匯出至Google文件

根據帳號類型選擇合適的分享方式，確保內容完整傳遞。

注意事項

  • 確認帳號類型以選擇分享方式

特殊情況處理

  • 如果無法直接分享對話，改用Google文件匯出並設定權限。

提升AI批改作文的針對性與實用性。

我是國中會考的作文批改老師，請你依照圖片當中的文章，給他三個優點，兩個寫作建議，跟一個參考性的分數。

  • 設定AI角色
  • 明確列出批改要求

注意事項

  • 引導語需具體明確

特殊情況處理

AI在國中作文批改的應用

利用AI（如Gemini、康軒AI高手）協助國中作文批改，能自動讀取學生作文內容，給予三個優點、兩個寫作建議及一個參考性分數，並可生成範文供教師參考。

關鍵點

  • AI可讀取圖片或PDF檔案中的作文內容
  • 可自動生成優點、建議與分數
  • 可根據教師回饋調整回應內容
  • 可生成範文供教師參考

教師只需拍照上傳作文，AI會自動分析內容並給出三個優點、兩個建議及一個參考分數。教師可針對AI建議進行追問或要求生成範文。AI具備記憶功能，能根據教師的偏好調整回饋內容。若作文有兩面，部分平台（如康軒AI高手）可支援多張圖片上傳，Gemini目前僅支援一張圖片，需轉為PDF檔案處理。

實例

教師將學生雙面作文正反面拍照上傳至康軒AI高手平台，平台可辨識並批改兩面內容，給予分數與回饋。

  • 教師將作文正反面分別拍照
  • 在平台新增頁面上傳兩張圖片
  • AI自動分析並給出回饋與分數

注意事項

  • AI批改結果需教師審核後再給學生
  • AI回饋內容會根據教師互動逐漸個人化

特殊情況處理

  • 如果遇到作文超過一頁，Gemini僅支援一張圖片，建議將作文轉為PDF檔案再上傳。
  • 如果AI辨識錯誤，可針對特定段落單獨拍照上傳。

AI平台功能與付費差異

不同AI平台（如Gemini、康軒AI高手）在圖片上傳數量、功能完整度及是否需付費等方面有所差異。

關鍵點

  • Gemini目前僅支援一張圖片上傳
  • 康軒AI高手可支援多張圖片（雙面作文）
  • 康軒AI高手因有付費，功能較完整
  • AI平台功能會隨時間逐步增強

Gemini目前僅能處理一張圖片，若作文有多頁需轉為PDF檔案。康軒AI高手因有付費，支援多張圖片上傳，適合批改雙面作文。AI平台功能會隨技術進步逐步提升，建議教師可先使用免費功能，待功能成熟再考慮付費。

實例

Gemini僅支援單張圖片，康軒AI高手可支援多張圖片並有付費功能，教師可根據需求選擇平台。

  • 教師根據作文頁數選擇合適平台
  • 如需多頁批改，優先考慮康軒AI高手

注意事項

  • 教師不需急於購買AI服務，免費功能已足夠日常使用
  • AI平台功能會持續更新，建議觀察發展再決定是否付費

特殊情況處理

  • 如果平台尚未支援多圖片，建議將多頁內容合併為PDF檔案上傳

AI記憶功能與個人化回饋

AI平台具備記憶功能，能根據教師的互動與回饋調整生成內容，提升個人化程度。

關鍵點

  • AI會記住教師的偏好與常用回饋
  • 教師可透過給予大拇指或大讚調整AI回應
  • AI生成內容會越來越貼近教師需求

教師與AI互動越多，AI會學習教師的評語風格與偏好，未來生成的回饋會更符合教師需求。教師可透過平台的評價功能（如大拇指、大讚）反饋AI回應品質。

注意事項

  • 教師需多與AI互動以提升個人化回饋品質

教師在使用AI批改時的審核責任

AI批改僅供參考，教師需審核AI回饋內容後再給學生，確保回饋品質與正確性。

關鍵點

  • AI回饋內容可能有誤，需教師審核
  • 教師應根據自身專業判斷調整AI建議

AI生成的批改建議與分數僅供參考，教師需根據自身專業經驗審核內容，避免錯誤資訊傳遞給學生。

注意事項

  • AI回饋需經教師審核後才能給學生

Felo專業搜尋平台介紹與操作

Felo是一個日本開發的專業AI搜尋平台，能協助教師快速搜尋教學資料，支援學術、新聞等多種來源，每日免費用戶可進行五次深入研究。

關鍵點

  • Felo可取代Google進行教學資料搜尋
  • 支援學術、新聞（X/Twitter）等多種來源
  • 每日免費用戶可進行五次深入研究
  • 需帳號登錄，建議用Google帳號

教師只需登錄Felo平台，輸入關鍵字即可自動搜尋並彙整多個來源的資料，節省備課時間。平台支援學術、新聞等多種來源，適合不同教學需求。建議教師用Google帳號登錄並開啟同步，方便資料記錄與跨裝置使用。

實例

教師在Felo平台輸入『登冠雀樓』，選擇研究模式，平台自動搜尋並彙整相關資料，支援學術與新聞來源。

  • 登錄Felo平台
  • 輸入關鍵字『登冠雀樓』
  • 選擇研究模式並送出
  • 平台自動搜尋並彙整多個來源資料

注意事項

  • 需帳號登錄並開啟同步以保存操作記錄
  • 免費用戶每日有五次深入研究次數限制

    • 限制：規範AI生成內容的範圍與格式
    • 追問：針對生成內容進一步詢問細節

    Google帳號與Gemini平台登入操作

    建議老師們使用Google帳號（如Gmail）登入Gemini平台，教育部Go帳號與臺北市帳號也可能適用。登入後可於gmail.google.com進行操作，付費版與非付費版僅有彩色符號等小差異，初學者建議使用非付費版。

    關鍵點

    • Google帳號、Gmail帳號皆可登入Gemini
    • 教育部Go帳號、臺北市帳號需實際測試
    • 付費版多彩色符號，功能差異不大
    • 初學者不需付費

    老師們可依照自身帳號類型選擇登入方式，若遇到無法登入的情況，建議嘗試不同帳號。

    注意事項

    • 確認帳號類型是否支援Gemini登入
    • 初期不建議付費

    特殊情況處理

    • 如果臺北市帳號無法登入，建議改用Gmail或教育部Go帳號。

    Gemini圖片生成與模組選擇

    Gemini平台可根據指令生成圖片，常用2.0模組，2.5模組雖號稱最聰明但有次數限制。生成圖片時需明確指令，避免誤生成照片。

    關鍵點

    • 2.0模組無次數限制，適合一般使用
    • 2.5模組有次數限制，適合進階需求
    • 指令需明確指定生成『圖片』而非『照片』

    根據需求選擇模組，並在指令中明確說明生成內容，避免系統誤判。
